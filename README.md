## 🟫 プロジェクト概要

『佐々木日記』は、指定被験体「佐々木（仮称：Ks-0.GuaKi）」の挙動、および彼が所有する文房具の流動的遷移（紛失・略奪等）を記録するための、高精度・非干渉型観測プラットフォームです。

単なる日記アプリではなく、観測者が透明な壁の向こう側から対象を記述する「研究ログ」としてのインターフェースを備えています。

## 🟫 特徴的な機能

### 🟫 リキッド・トランジション (Sasaki Liquid Transition)
ページ切り替え時に発生する、上下から波打つような独自の液体アニメーション。情報の「流動性」を視覚的に表現し、ユーザーを非日常的な観測体験へと誘います。

### 🟫 観測手帳（ドロワーメニュー）
`ESC`キーまたはアイコンから展開されるサイドメニュー。主要な登場人物（佐々木、大竹、観測者）のプロファイルや、重要な観測記録へのクイックアクセスが可能です。

### 🟫 運営通達システム
事務所からの緊急指令やシステムアップデートをリアルタイムに表示。
- `SYSTEM`: 基幹システムの更新
- `ALERT`: 被験体に関する緊急事態
- `POWER`: 情報操作権の授与に関する通知

### 🟫 動態検索エンジン
ハッシュタグ（#佐々木 #文房具 等）やキーワードによる高速フィルタリング。膨大なログから、佐々木が「強がった瞬間」を逃さず特定します。

## 🟫 デザイン・スタック

本プロジェクトは、手書きの温かみと、研究機関としての冷徹な機能美を両立させるために以下の技術・アセットを使用しています。

| 要素 | 指定内容 |
| :--- | :--- |
| **Fonts** | `Zen Kurenaido` (メイン), `Yomogi` (サブ/手書き風) |
| **Colors** | `#fcf9f2` (紙質色), `#4e3629` (インク色), `#ffab91` (アクセント) |
| **UI Design** | 不規則な`border-radius`による「付箋・手書き感」の演出 |
| **Animation** | `Cubic-bezier`を用いた有機的な挙動 |

## 🟫 技術仕様

外部ライブラリに依存しない**Pure Vanilla Stack**で構築されています。

- **Frontend:** HTML5, CSS3 (Custom Variables, Grid/Flexbox)
- **Logic:** Vanilla JavaScript (ES6+)
- **Architecture:** State-based Rendering & Pseudo-routing
- **Responsiveness:** モバイル端末に完全最適化（グリッドの自動調整機能搭載）
